# 我的第一篇博客

博客终于上线了！

## 为什么要写博客？

作为一名 AI 产品经理，日常工作中会接触到大量有趣的技术和产品思考。我希望通过写博客来：

- **沉淀思考**：把零散的想法整理成系统的文章
- **分享经验**：将工作中踩过的坑和积累的经验分享出来
- **持续学习**：写作本身就是最好的学习方式

## 计划写些什么

未来打算围绕以下几个方向来写：

### AI 产品
- AI Agent 的产品设计思考
- Chatbot 用户体验优化
- 大模型在 C 端产品中的应用

### 技术探索
- RAG 系统的实践经验
- Prompt Engineering 技巧
- 新模型能力评估

### 产品方法论
- 数据驱动的产品决策
- A/B Testing 实践
- 用户增长策略

## 技术栈

这个博客使用纯静态方式搭建：

```
blog/
  posts.json        # 文章元数据
  posts/
    hello-blog.md   # Markdown 文章
```

写一篇新文章只需要两步：
1. 在 `posts/` 目录下写一个 `.md` 文件
2. 在 `posts.json` 中添加一条记录

简单直接，够用就好。
